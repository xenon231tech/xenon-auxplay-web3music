# Changelog

All notable changes to Auxplay will be documented in this file.

## [1.1.0] - 2026-05-18

### Added
- 🔍 Search modal for songs and artists
- 🌟 Top creators section with follow functionality
- 🎧 Trending playlists with add to queue
- 💡 How It Works educational section
- 💰 Enhanced tip modal with custom amounts
- 🔔 Toast notifications for all actions
- 📱 Improved mobile responsiveness
- 🎨 Better animations and transitions

### Changed
- Refactored JavaScript for better maintainability
- Improved CSS organization with Tailwind
- Better modal animations
- Updated README with more details

### Fixed
- Modal close functionality
- Responsive design issues on smaller screens
- Toast notification positioning

## [1.0.0] - 2026-05-18

### Added
- ✨ Initial release
- 🎵 Featured tracks dashboard
- 💻 Interactive UI with Tailwind CSS
- 🎧 Music player bar with controls
- 📊 Analytics sidebar (desktop)
- 💝 Tip/Donate functionality
- 🎨 Beautiful dark mode theme
- 📱 Mobile responsive design
- 🔗 Web3 wallet button placeholder
- 🎬 Hero section with CTAs

### Technical
- Vanilla JavaScript (no dependencies)
- Tailwind CSS for styling
- Font Awesome icons
- HTML5 semantic markup
